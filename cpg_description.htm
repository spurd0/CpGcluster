DNA Sequence

DNA Sequence should be in FASTA format, sample sequence:
>seq
aaaacatga

d

The threshold distance on basis of a given percentile
For example: d=25 calculates the percentile 25 of the genomic
CpG distance distribution and takes this value as the threshold
distance
The recommended value is 50 (median distance)

P-value

The maximal P-value under which a CpG cluster is considered as a
CpG island
The recommended limit is 1E-5
